One function of an operating system is to control system performance. The OS periodically monitors over the system to ensure healthy performance. It will continue to monitor and record operation times and response to have a complete view of system health. This information is used to increase performance.
Another function of an operating system is memory management. Operating systems manages Primary memory and Main memory in order for the computer to operate properly. When programs begin and ends the memory needed to use those programs get allocated accordingly based on whats being used. The operating system will allocate memory when a program is being used and also deallocates memory when a program terminates. 
Another function of an operating system is file management. It is because of the operating system that we can browse and make use of our files in an efficient manner. The OS stores and organizes our files based on our preferences and provide an easy interface to command and edit files.
